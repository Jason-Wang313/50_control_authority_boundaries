# Control Authority Boundaries in Shared Autonomy

Submission-hardening version: v2.

Decision: workshop-only.

The paper has a useful diagnostic thesis: authority transfer should expose physical phase, input-quality, and risk boundaries instead of relying only on intent confidence or elapsed time. The v2 hardening pass narrows the claim. A threshold baseline tuned on 396 training cases reaches 1.000 holdout success and 0.000 unsafe ceding on the 204-case holdout split, beating the hand-coded authority-boundary rule's 0.882 holdout success.

This repository should not claim algorithmic superiority or real-robot safety. It is a concept-and-diagnostic workshop paper unless future evidence adds real interaction traces, learned boundary extraction, and stronger tuned shared-autonomy baselines.
